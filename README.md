# Generational-health-drift-systematic-review
This repository contains supplementary material for a systematic review of health and morbidity comparisons across the British birth cohort studies at similar ages. 

**Working paper:** Gimeno L, Moreno-Agostino D, Danka M, Guo Y, Goisis A, Dowd JB and Ploubidis GB (2025). The Generational Health Drift: A Systematic Review of Evidence from the British Birth Cohort Studies. *MedRxiv*. doi: [10.1101/2025.06.12.25329414](https://doi.org/10.1101/2025.06.12.25329414)
